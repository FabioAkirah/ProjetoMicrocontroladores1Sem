# ProjetoMicrocontroladores1Sem
Descrição do projeto -
O projeto consiste em criar um sistema para automatizar a ação de regar uma planta.
Além de realizar a função de regar a planta quando necessário (umidade da terra relativamente baixa), a ideia é que o usuário possa também monitorar em tempo real algumas outras coisas como umidade da terra, temperatura ambiente e luminosidade na planta. Algumas dessas informações ficariam disponíveis num LCD integrado ao vaso, ou poderiam ser obtidas no celular via bluetooth.

Módulos –
Timer
PIO
ADC
Bluetooth
